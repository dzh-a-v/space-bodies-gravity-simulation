# Программа

Исходный код должен находиться в `programs/`. Программа должна быть написана на языке `Python`.

# .gitignore

Личные заметки не пушить (для этого их надо добавить в файл `.gitignore`).

# Промпты

Все промпты надо писать в `docs/prompts/prompts.md`. Записывать надо и запрос, и ответ.

Даже если промпт неудачен, его всё равно надо туда записать.

Разграничивать промпты горизонтальными линиями (3 нижних подчеркивания): между запросом и промптом одна линия, между двумя разными промптами две линии. Формат:
```
запрос
___
ответ
___
___
запрос
___
ответ
___
___
запрос
___
ответ
___
___
```
